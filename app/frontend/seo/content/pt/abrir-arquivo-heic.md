# Como abrir arquivo HEIC: todos os métodos para Windows 11, Windows 10 e Mac

Você copiou as fotos do iPhone para o computador, deu dois cliques — e o Windows pediu um aplicativo ou tentou vender um codec pago. ## O que é um arquivo HEIC — em duas linhas

HEIC significa *High Efficiency Image Container*. A Apple usa o formato desde 2017 porque ele guarda as fotos com a mesma qualidade ocupando **metade do espaço** de um JPG. O problema: o Windows e muitos programas não conseguem exibir HEIC sem software adicional. A explicação completa está no nosso guia [O que é um arquivo HEIC?](/pt/what-is-heic)

## Como abrir arquivo HEIC no Windows 11

O Windows 11 consegue abrir arquivos HEIC, em muitos casos, direto no aplicativo **Fotos**. Se não funcionar, estão faltando as extensões HEIF:

1. Abra a **Microsoft Store** e procure por "Extensões de Imagem HEIF".
2. Instale a extensão gratuita da Microsoft.
3. Dê dois cliques no arquivo HEIC — agora ele abre no aplicativo Fotos.

**Atenção:** para vídeos, a Microsoft exige também as "Extensões de Vídeo HEVC" (pagas). Para fotos, as extensões HEIF gratuitas bastam.

Se a Store não estiver disponível (em computadores corporativos, por exemplo), use o [visualizador HEIC gratuito no navegador](/pt/heic-viewer) — não precisa de permissão de administrador.

## Como abrir arquivo HEIC no Windows 10

No Windows 10 a situação é a mesma, só que falha com mais frequência: o aplicativo Fotos só exibe HEIC depois de instalar as **Extensões de Imagem HEIF** da Microsoft Store — e em versões antigas do Windows 10 nem isso funciona direito.

- **Visualizadores com suporte a HEIC:** IrfanView ou XnView exibem HEIC depois de instalar os plugins adequados.

## Como abrir arquivo HEIC no Mac

No Mac é fácil: o **Pré-Visualização** e o **Fotos** abrem arquivos HEIC nativamente desde o macOS High Sierra (2017). Basta dar dois cliques.

A coisa complica quando você quer **compartilhar** as fotos — com usuários de Windows, com algum órgão público ou com um formulário web que só aceita JPG. Nesse caso: [converter HEIC para JPG no Mac](/pt/how-to-convert-heic-to-jpg-mac) ou transforme direto no navegador.

## Evitar HEIC no futuro: configurar o iPhone para salvar JPG

Se quiser resolver o problema pela raiz, configure o iPhone para capturar direto em JPG:

1. **Ajustes** → **Câmera** → **Formatos**
2. Escolha **"Mais compatível"** em vez de "Alta eficiência".

A partir daí a câmera salva JPG. Lembre que arquivos JPG ocupam cerca do dobro do espaço — em um iPhone cheio isso pode pesar.

## Perguntas frequentes

### Por que o Windows não abre arquivos HEIC?

HEIC é um formato contêiner usado pela Apple, baseado em HEIF/HEVC. A Microsoft não inclui os codecs necessários de fábrica: ela os oferece como extensões separadas na Store — e a variante HEVC, ainda por cima, é paga. Por isso o duplo clique falha em um sistema recém-instalado.

### É seguro abrir no navegador?

Sim — no OpenMyHEIC os arquivos **não são enviados** para lugar nenhum. A decodificação roda com WebAssembly/JavaScript direto no seu dispositivo. Você pode testar desconectando a internet: o visualizador continua funcionando.

### Perco qualidade ao converter para JPG?

Muito pouca. O JPG, assim como o HEIC, é um formato com perdas; no nível de qualidade padrão a diferença é praticamente invisível a olho nu. Se quiser qualidade máxima, converta para [PNG](/pt/heic-to-png) — sem perdas, mas com arquivos maiores.

### Posso abrir e converter vários arquivos HEIC de uma vez?

Sim. O [conversor em lote](/pt/batch-convert) processa pastas inteiras de fotos HEIC de uma só vez e devolve um ZIP com todos os JPGs.
