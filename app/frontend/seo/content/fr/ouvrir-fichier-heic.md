# Ouvrir un fichier HEIC : toutes les méthodes (Windows 11, Windows 10, Mac)

Vous avez transféré des photos de votre iPhone vers votre PC, double-cliqué sur l'une d'elles… et Windows vous demande de choisir une application, voire d'acheter un codec. La raison : la photo est un **fichier HEIC**, le format photo par défaut d'Apple depuis iOS 11. Bonne nouvelle : on peut **ouvrir un fichier HEIC gratuitement** sur n'importe quel système — et ce guide passe en revue toutes les méthodes : Windows 11, Windows 10, Mac, et une solution dans le navigateur qui fonctionne partout.

## Un fichier HEIC, c'est quoi exactement ?

HEIC signifie *High Efficiency Image Container*. Apple l'utilise depuis 2017 parce qu'il stocke les photos avec la même qualité qu'un JPG mais en occupant **environ deux fois moins d'espace**. Le problème : Windows et beaucoup de logiciels ne savent pas l'afficher sans module complémentaire. Pour tout comprendre sur ce format, consultez notre guide [Qu'est-ce qu'un fichier HEIC ?](/fr/what-is-heic)

## Pourquoi mes photos sont en HEIC ?

Parce que votre iPhone (ou iPad) est réglé sur le format « Haute efficacité », activé par défaut depuis iOS 11. Chaque photo prise avec l'appareil photo est alors enregistrée en HEIC au lieu de JPG. Apple économise ainsi la moitié de l'espace de stockage — mais crée des soucis de compatibilité dès que les photos sortent de l'écosystème Apple.

Pour que votre iPhone photographie directement en JPG :

1. **Réglages** → **Appareil photo** → **Formats**
2. Choisissez **« Le plus compatible »** au lieu de « Haute efficacité ».

Les nouvelles photos seront en JPG (fichiers environ deux fois plus lourds). Les photos déjà prises restent en HEIC — pour celles-là, voyez la conversion ci-dessous.

## Ouvrir un fichier HEIC sur Windows 11

Windows 11 peut souvent afficher les fichiers HEIC dans l'application **Photos**. Si ce n'est pas le cas, il manque les extensions HEIF :

1. Ouvrez le **Microsoft Store** et cherchez « Extensions d'images HEIF ».
2. Installez l'extension gratuite de Microsoft.
3. Double-cliquez sur votre fichier HEIC : il s'ouvre désormais dans Photos.

**Attention :** pour les vidéos, Microsoft facture les « Extensions vidéo HEVC » (0,99 €). Pour les photos, l'extension HEIF gratuite suffit.

Si le Store est bloqué (ordinateur d'entreprise, par exemple), utilisez la [visionneuse HEIC dans le navigateur](/fr/heic-viewer) — aucun droit administrateur nécessaire.

## Ouvrir un fichier HEIC sur Mac

Sur Mac, c'est simple : **Aperçu** et **Photos** ouvrent les fichiers HEIC nativement depuis macOS High Sierra (2017). Un double-clic suffit.

Le besoin apparaît plutôt au moment de **partager** — avec un utilisateur Windows, une administration ou un formulaire web qui n'accepte que le JPG. Dans ce cas : [convertir HEIC en JPG sur Mac](/fr/how-to-convert-heic-to-jpg-mac) ou conversion directe dans le navigateur.

## Questions fréquentes

### Pourquoi Windows ne peut-il pas ouvrir les fichiers HEIC ?

HEIC est un format conteneur basé sur HEIF/HEVC utilisé par Apple. Microsoft ne fournit pas les codecs nécessaires par défaut : ils sont proposés en extensions séparées dans le Store, dont une payante (HEVC). D'où l'échec du double-clic sur un système fraîchement installé.

### Ouvrir un HEIC dans le navigateur, est-ce sûr ?

Oui — avec OpenMyHEIC, les fichiers **ne sont pas téléversés**. Le décodage s'exécute en WebAssembly/JavaScript directement sur votre appareil. Vous pouvez le vérifier : coupez la connexion Internet, la visionneuse fonctionne toujours.

### Perd-on en qualité en convertissant en JPG ?

Très peu. Le JPG est, comme le HEIC, un format avec perte ; au niveau de qualité standard, la différence est invisible à l'œil nu. Pour une qualité maximale, convertissez plutôt en [PNG](/fr/heic-to-png) — sans perte, mais avec des fichiers plus lourds.

### Puis-je ouvrir et convertir beaucoup de fichiers HEIC à la fois ?

Oui. Le [convertisseur par lots](/fr/batch-convert) traite des dossiers entiers de photos HEIC en une seule fois et renvoie un ZIP avec toutes les images converties.
