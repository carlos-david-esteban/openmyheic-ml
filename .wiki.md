# Project Summary
**OpenMyHEIC.com** is a multilingual web application that converts HEIC files, commonly used by Apple devices, into various formats such as JPG, PNG, WebP, PDF, GIF, and BMP. The application delivers a fast, private, and user-friendly experience by executing all conversions client-side without requiring file uploads. It is optimized for SEO and provides informative content about the HEIC format and conversion techniques, now supporting 11 languages for enhanced accessibility.

# Project Module Description
The application consists of several functional modules:
- **Homepage**: Introduction to the converter with an integrated HEIC-to-JPG feature.
- **Conversion Pages**: Dedicated pages for each format conversion (e.g., HEIC to JPG, PNG, WebP, etc.).
- **Batch Conversion**: Enables users to upload and convert multiple HEIC files simultaneously.
- **Blog/SEO Pages**: Informative articles about HEIC format and conversion techniques to enhance SEO.
- **User Interface Components**: Reusable components for layout, buttons, sliders, and alerts.
- **Shared Head System**: Manages dynamic meta tags across pages using `react-helmet-async`.
- **Language Switcher**: Allows users to easily switch between supported languages.
- **Legal Pages**: Pages for Privacy Policy, Terms of Use, and Contact, available in all supported languages.

# File Description Inventory
- **README.md**: Project overview and setup instructions.
- **.env.example**: Template for environment variables with documentation.
- **.env**: Local environment variables file with placeholder values.
- **index.html**: Main HTML file for the application, includes static meta tags and AdSense script.
- **package.json**: Project dependencies and scripts.
- **src/**: Contains all source files including components, pages, and utilities.
- **components/**: Reusable UI components such as converters, layout, and shared head.
- **pages/**: Individual page components for different functionalities, including legal pages.
- **lib/**: Utility functions and API integrations.
- **seo-scripts/**: Scripts for generating SEO content and sitemaps.
- **i18n/**: Contains internationalization configuration and language translation files.

# Technology Stack
- **Framework**: React + Vite with React Router
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Client-Side Conversion Library**: heic2any
- **PDF Generation**: jsPDF
- **ZIP File Handling**: JSZip
- **Image Processing**: gif.js
- **SEO Management**: react-helmet-async

# Usage
1. **Install Dependencies**:
   ```
   pnpm install
   ```
2. **Build the Project**:
   ```
   pnpm run build
   ```
3. **Run the Application**:
   ```
   pnpm run start
   ```
